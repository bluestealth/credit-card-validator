Welcome to Credit Card Validator

This is a JavaScript credit card number validator for Visa, American Express, Discover, Mastercard, and JCB.

To checkout this project using git use the following commands.

$ git clone git@github.com:bluestealth/credit-card-validator.git
$ cd credit-card-validator

To use in your node project just use the following command.

$ npm install --save credit-card-validator
